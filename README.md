# Download

```
git clone https://github.com/wh1tesh1t/xashbox
unzip xashbox.zip
cd xashbox
```
```
cp -r * ../YOUR_GAME_DIR
```
or
```
mv * ../YOUR_GAME_DIR
```


### What is it?
This is **entity spawn** menu for Half-Life which uses **Entity Tools** feature from Xash3D FWGS


### How I can use this?

You must make a touch button with the command:
```
exec xashbox/my_menu
```
```
exec xashbox/menu
```
```
xashbox_menu
```
